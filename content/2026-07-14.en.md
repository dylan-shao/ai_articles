## GitHub Code Quality license estimate in public preview - GitHub Changelog
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-07-13
- URL: https://github.blog/changelog/2026-07-13-github-code-quality-license-estimate-in-public-preview/

GitHub added a license cost estimate for Code Quality ahead of paid GA, letting enterprises view projected monthly spend based on active committers. The estimate only covers per-committer license cost and excludes GitHub Actions minutes consumed by CodeQL as well as usage-based charges for AI features such as Copilot Autofix.

Why it matters: For platform and engineering leads, this is primarily a budgeting and governance signal: GitHub is productizing code quality/security workflows with per-active-committer pricing. While technically shallow, it directly informs whether to enable it org-wide, restrict it to critical repos, or narrow pilots before GA.

Key takeaways:
- GA pricing is $10 per active committer per month, and the estimate uses standard list pricing without discounts.
- Total cost should not be inferred from license alone: teams must also model CodeQL Actions minutes and usage-based AI remediation charges.
