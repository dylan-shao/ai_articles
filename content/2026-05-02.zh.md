## Upcoming deprecation of GPT-5.2 and GPT-5.2-Codex - GitHub Changelog
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-05-01T00:00:00+00:00
- URL: https://github.blog/changelog/2026-05-01-upcoming-deprecation-of-gpt-5-2-and-gpt-5-2-codex/

GitHub 宣布将于 2026-06-01 在大多数 Copilot 体验中弃用 GPT-5.2 与 GPT-5.2-Codex，并分别建议迁移到 GPT-5.5 与 GPT-5.3-Codex。例外是 GPT-5.2-Codex 在 Copilot Code Review 中暂时保留。企业管理员还需要确认替代模型已通过模型策略开放给用户与团队。

Why it matters: 对工程平台团队而言，这是一条典型但高影响的供应商模型生命周期事件。凡是依赖 Copilot 指定模型、做了内部提示词调优、评测基线、开发者培训或合规审批的组织，都需要提前验证替代模型在聊天、补全、agent 模式与代码审查中的行为差异，避免开发体验、成本与输出质量发生隐性回归。

Key takeaways:
- 模型弃用不仅影响交互式聊天，也影响 inline edits、ask/agent modes、code completions 等多种 Copilot 工作流。
- 企业管理员需要检查 Copilot model policies，确保 GPT-5.5 与 GPT-5.3-Codex 已对目标用户开放，否则迁移会在组织层面被策略阻断。
