## AI credits consumed per user now in the Copilot usage metrics API
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-06-19
- URL: https://github.blog/changelog/2026-06-19-ai-credits-consumed-per-user-now-in-the-copilot-usage-metrics-api/

GitHub 在 Copilot usage metrics API 中新增 ai_credits_used 字段，可在企业和组织级的 1 天与 28 天用户报表中查看每个用户每日消耗的 AI credits 总量。该指标与现有使用报表放在一起，便于把消耗、活跃度和采用情况关联分析，但目前仍是用户总量口径，不能按功能、模型或交互 surface 拆分。

Why it matters: 对负责平台、开发效率和预算控制的团队来说，这是把 AI 编码从“是否有人在用”推进到“谁在消耗、消耗是否匹配价值”的关键观测点。它使组织能够开始做 Copilot 成本归因、团队采用差异分析、异常消耗监控，以及 usage-based billing 的预算预测。

Key takeaways:
- 新增 ai_credits_used 字段，覆盖 enterprise/org 级 users-1-day 与 users-28-day 用户报表。
- 该指标基于 AI credits consumption 数据，但不是计费账单明细，不能直接当作 billed total 使用。
- 当前只提供每用户总量，尚未细分到具体功能、模型或使用入口，这限制了精细化优化。
