## IP allow list coverage for EMU namespaces in general availability - GitHub Changelog
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-06-08
- URL: https://github.blog/changelog/2026-06-08-ip-allow-list-coverage-for-emu-namespaces-in-general-availability/

GitHub 宣布 Enterprise Managed Users（EMU）命名空间的 IP allow list 覆盖进入 GA。企业现在可以把原生 IP allow list 策略统一施加到 EMU 用户命名空间下的仓库访问，覆盖 Web UI、Git 协议、API，以及 PAT、应用令牌和 SSH key 等凭据。

Why it matters: 虽然这不是 AI 编码能力本身，但对大规模企业导入 Copilot、编码代理和自动化开发工作流很关键：当开发活动越来越多地通过 API、CLI、代理和服务账号执行时，网络边界与凭据使用范围必须能被统一治理。该能力降低了“用户个人命名空间成为治理盲区”的风险。

Key takeaways:
- EMU 用户命名空间现在可受企业级 IP allow list 统一约束，不再只局限于组织层或部分资源。
- 控制面覆盖 Web、Git、API 与多类凭据，意味着对代理式开发和自动化工具链的访问路径也能纳入同一网络策略。
