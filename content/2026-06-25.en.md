## Self-service credential revocation for incident response - GitHub Changelog
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-06-24
- URL: https://github.blog/changelog/2026-06-24-self-service-credential-revocation-for-incident-response/

GitHub added break-glass self-service credential revocation for Enterprise, allowing authorized admins to bulk revoke SSO authorizations and, in some cases, delete tokens and SSH keys across a user or the whole enterprise. Individual users also get a self-service flow to revoke their own credentials, with audit-log and email visibility.

Why it matters: For orgs rolling out AI coding assistants, CLI agents, and automated developer workflows, credential sprawl and delegated execution increase blast radius during incidents. This isn’t an AI-coding breakthrough, but it does strengthen an important operational control for containing compromised developer identities and tooling access.

Key takeaways:
- Supports enterprise-scale revocation of SSO authorizations and, for EMU scenarios, deletion of user tokens and SSH keys.
- Adds a user self-service one-click revocation flow plus audit-log visibility, improving incident response and compliance traceability.

Tags: security, enterprise-governance, developer-tooling, incident-response
Scores: signal 6/5, novelty 4/5, actionability 7/5

## Changes to model selection for Free and Student plans - GitHub Changelog
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-06-24
- URL: https://github.blog/changelog/2026-06-24-changes-to-model-selection-for-free-and-student-plans/

GitHub is simplifying Copilot Free and Student plans to use only auto model selection, removing manual model choice and retiring the Preview label for Microsoft-released models.

Why it matters: This is a product-direction signal: AI coding tools are moving from explicit model choice toward platform-managed routing per task. But because it targets only Free/Student plans and provides no routing mechanics, eval methodology, or quality data, the technical value is limited.

Key takeaways:
- Model selection is increasingly being abstracted into the platform rather than left to end users.
- The announcement does not disclose routing evaluation criteria, fallback strategy, or cost/quality trade-offs.

Tags: copilot, model-routing, product-direction
Scores: signal 4/5, novelty 3/5, actionability 3/5
