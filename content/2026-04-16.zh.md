## Enable Copilot cloud agent via custom properties - GitHub Changelog
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-04-15
- URL: https://github.blog/changelog/2026-04-15-enable-copilot-cloud-agent-via-custom-properties/

GitHub 允许企业管理员按组织选择性启用 Copilot cloud agent，而不再只能全开、全关或完全下放给组织自行决定。新功能支持通过 AI Controls 页面或 REST API 管理，并可结合 organization custom properties 进行初始筛选。

Why it matters: 这类能力直接影响 AI coding agent 在大企业中的落地速度。对平台与工程治理负责人来说，真正的难点通常不是“能不能用”，而是如何安全地做试点、限定范围、逐步扩张，并保留审计与策略控制。这次更新提供了一个更实际的 adoption control plane。

Key takeaways:
- 新增策略模式：可只对选定组织启用 Copilot cloud agent，支持渐进式 rollout。
- 提供 3 个 REST API 端点用于设定策略、加入启用列表、移出启用列表，便于平台自动化接入。
- custom properties 只在配置时评估一次，后续属性变化不会自动重新计算启用状态，这对治理自动化设计很关键。
