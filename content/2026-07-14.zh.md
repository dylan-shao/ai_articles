## GitHub Code Quality license estimate in public preview - GitHub Changelog
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-07-13
- URL: https://github.blog/changelog/2026-07-13-github-code-quality-license-estimate-in-public-preview/

GitHub 在 Code Quality 正式收费前推出许可证费用预估功能，企业可在 Billing and licensing 页面查看基于 active committers 的预计月度成本。该预估仅覆盖每位活跃提交者的许可费用，不包含 CodeQL 所消耗的 GitHub Actions 分钟，也不包含 Copilot Autofix 等 AI 能力的用量费用。

Why it matters: 对平台负责人而言，这是一条预算与治理信号：GitHub 正把代码质量/安全能力进一步产品化并按活跃提交者收费。即使技术深度有限，该更新能直接支持是否在全组织开启、只对核心仓库启用，或在 GA 前收缩试点范围等决策。

Key takeaways:
- GA 定价为每位活跃提交者每月 10 美元，且预估值以标准标价计算，不含折扣。
- 总成本不能只看许可证：还要额外核算 CodeQL 的 Actions 分钟和 AI 修复类能力的用量费用。
