## Self-service credential revocation for incident response - GitHub Changelog
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-06-24
- URL: https://github.blog/changelog/2026-06-24-self-service-credential-revocation-for-incident-response/

GitHub 为 Enterprise 提供新的“break-glass”自助式凭证撤销能力，企业所有者或具备相应权限的成员可针对单个用户或全企业批量撤销 SSO 授权、删除令牌与 SSH key；个人用户也可在设置中一键撤销自己的全部相关凭证，并通过审计日志和邮件获取追踪信息。

Why it matters: 对正在推广 AI 编码助手、CLI agent 和自动化工作流的组织来说，凭证扩散与代理代表用户执行操作会显著提高事故处置复杂度。这项能力虽然不是 AI coding 本身的创新，但它补齐了企业在“工具被盗用/账户被接管”场景下的快速止血机制。

Key takeaways:
- 支持企业级批量撤销 SSO 授权，以及在 EMU 场景下删除用户 token 和 SSH key。
- 提供个人级一键自助撤销入口，并将结果写入审计日志，便于事件响应与合规追踪。

Tags: security, enterprise-governance, developer-tooling, incident-response
Scores: signal 6/5, novelty 4/5, actionability 7/5

## Changes to model selection for Free and Student plans - GitHub Changelog
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-06-24
- URL: https://github.blog/changelog/2026-06-24-changes-to-model-selection-for-free-and-student-plans/

GitHub 将 Copilot Free 与 Student 套餐的模型选择收敛为仅支持自动路由，用户不再手动选择模型；同时移除微软模型的 Preview 标签。

Why it matters: 这是一个产品策略信号：AI 编码工具正从“让开发者自己挑模型”转向“平台按任务自动路由模型”。但由于仅针对免费/学生计划，且缺少技术实现、评测方法和路由指标，参考价值有限。

Key takeaways:
- 模型选择体验正被抽象成平台层能力，而非终端用户决策。
- 公告没有公开自动路由的评估依据、失败回退策略或成本/质量权衡。

Tags: copilot, model-routing, product-direction
Scores: signal 4/5, novelty 3/5, actionability 3/5
