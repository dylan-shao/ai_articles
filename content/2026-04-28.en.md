## Copilot cloud agent starts 20% faster with Actions custom images
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-04-27
- URL: https://github.blog/changelog/2026-04-27-copilot-cloud-agent-starts-20-faster-with-actions-custom-images/

GitHub says Copilot cloud agent now starts more than 20% faster by using GitHub Actions custom images to prebuild the runner environment. The cloud agent spins up when users assign an issue to Copilot, start a task from the Agents tab, or mention @copilot in a pull request; this follows a previously announced 50% startup improvement in March.

Why it matters: For platform and engineering leaders, this highlights a key infrastructure truth about hosted coding agents: user experience depends not just on model quality, but on cold-start latency, image prewarming, dependency reuse, and task scheduling. As agents increasingly execute real repo tasks in the cloud, runtime environment engineering becomes a core differentiator.

Key takeaways:
- GitHub is using Actions custom images for Copilot cloud agent, indicating further convergence between coding agents and CI infrastructure.
- The repeated focus on startup improvements suggests that a major bottleneck for cloud agents is environment preparation overhead, not just model inference speed.
