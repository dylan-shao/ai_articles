## Upcoming deprecation of GPT-5.2 and GPT-5.2-Codex - GitHub Changelog
- Source: GitHub Changelog
- Author: N/A
- Publish date: 2026-05-01T00:00:00+00:00
- URL: https://github.blog/changelog/2026-05-01-upcoming-deprecation-of-gpt-5-2-and-gpt-5-2-codex/

GitHub announced that GPT-5.2 and GPT-5.2-Codex will be deprecated across most Copilot experiences on 2026-06-01, with recommended migrations to GPT-5.5 and GPT-5.3-Codex respectively. The exception is that GPT-5.2-Codex remains temporarily available in Copilot Code Review. Enterprise admins may need to explicitly enable the replacement models through Copilot model policies.

Why it matters: For engineering platform teams, this is a standard but high-impact vendor model lifecycle event. Any organization that depends on specific Copilot models for prompts, eval baselines, developer workflows, or governance needs to validate replacement behavior across chat, completions, agent mode, and code review before the cutoff to avoid silent regressions in quality, UX, or cost.

Key takeaways:
- The deprecation affects not just chat, but also inline edits, ask/agent modes, and code completions across Copilot workflows.
- Enterprise admins should verify Copilot model policies so GPT-5.5 and GPT-5.3-Codex are actually available to target users; otherwise migration can fail at the policy layer.
